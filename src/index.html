How it works: A random key is generated and stored in a Move smart contract on Aptos. The contract releases it only once the chain's own clock passes your unlock date — that check happens on-chain, not in this page's JavaScript. Your message is AES-256 encrypted with that key and uploaded as a real blob to ShelbyNet. If you set a recipient, their wallet's signature is also mixed into the key, so only they can ever decrypt it.
